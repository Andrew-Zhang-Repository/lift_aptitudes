### Lift Aptitudes

#### Summary
Gym app where you can see how you compare with the gym standard with seeded data from strengthlevel.com. Used Next.js + Typescript as the framework and language, supabase for the postgres server, and Prisma ORM for postgres querying logic and for seeding.

---
#### Vercel Link
[Lift Aptitudes](https://lift-aptitudes.vercel.app/)
